# v0.9.0 — Organization Rename

Breaking release with module path rename from `agentplexus/go-elevenlabs` to `plexusone/elevenlabs-go`.

## Highlights

- Organization rename from agentplexus to plexusone with updated module naming convention

## Breaking Changes

| Component | Before | After |
|-----------|--------|-------|
| Go module | `github.com/agentplexus/go-elevenlabs` | `github.com/plexusone/elevenlabs-go` |

## Migration Guide

Update your import paths:

```go
// Before
import "github.com/agentplexus/go-elevenlabs"

// After
import "github.com/plexusone/elevenlabs-go"
```

Update your `go.mod`:

```bash
go mod edit -droprequire github.com/agentplexus/go-elevenlabs
go get github.com/plexusone/elevenlabs-go@v0.9.0
go mod tidy
```
