# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html),
and this changelog is generated by [Structured Changelog](https://github.com/grokify/structured-changelog).

## [0.5.0] - 2026-01-04

### Added

- OmniVoice provider integration (`omnivoice/` subpackage)
- `omnivoice/tts` - TTS provider implementing `tts.Provider` and `tts.StreamingProvider`
- `omnivoice/stt` - STT provider implementing `stt.Provider` and `stt.StreamingProvider`
- `omnivoice/agent` - Agent provider implementing `agent.Provider` with session management
- `examples/debug-test` - HTTP debugging utilities for API troubleshooting
- `examples/simple` - Basic TTS generation example

## [0.4.0] - 2024-12-31

### Added

- Voice settings presets for platforms: Udemy, Coursera, EdX, Instagram, TikTok, YouTube, Podcast, Audiobook
- Integration tests for null handling validation
- Audio utilities: `PCMBytesToWAV`, `PCMToWAV`, `ParsePCMSampleRate`
- `ParseAPIError()` function for detailed error extraction
- `IsForbiddenError()` helper function
- Output format validation for TTS requests
- `ogen-tools` post-processing in code generation pipeline

### Fixed

- JSON decoding errors when API returns `null` for nullable `$ref` fields (affects 205+ fields)

## [0.3.0] - 2024-12-28

### Added

- WebSocket TTS - Real-time text-to-speech streaming with word-level alignment
- WebSocket STT - Real-time speech-to-text with partial results and confidence scores
- Speech-to-Speech - Voice conversion with background noise removal
- Twilio Integration - Phone call integration for voice agents
- Phone Number Management - Manage phone numbers for agents
- `gorilla/websocket` dependency for WebSocket support
- 4 new documentation pages for real-time services
- Release notes documentation

### Changed

- Updated API coverage to ~75 methods
- Updated README with real-time service examples

## [0.2.0] - 2024-12-28

### Added

- MkDocs documentation site - 28 pages with Material theme
- `cmd/ttsscript` - Command-line tool for TTS script processing
- LMS/Udemy course production guide
- Pronunciation rules guide
- TTS script authoring guide
- API coverage tracking with method-level details

### Changed

- Repository transferred from `grokify/go-elevenlabs` to `agentplexus/go-elevenlabs`
- Updated all internal import paths
- Updated documentation URLs and references

## [0.1.0] - 2024-12-28

### Added

- Initial release of the ElevenLabs Go SDK
- 15 service wrappers: Text-to-Speech, Speech-to-Text, Voices, Voice Design, Sound Effects, Music, Audio Isolation, Forced Alignment, Text-to-Dialogue, Projects, Pronunciation Dictionaries, Dubbing, History, Models, User
- `ttsscript` package - TTS script authoring utilities
- `voices` package - Voice constants and metadata
- Functional options pattern for client configuration
- Automatic API key from environment variable
- Comprehensive error handling
- Multiple output format support (MP3, PCM, μ-law)
