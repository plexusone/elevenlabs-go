## [v0.8.0] - 2026-02-13

### Highlights

- New Cobra-based CLI (`elevenlabs`) with `tts` and `ttsscript` subcommands for text-to-speech generation
- YAML configuration file support with presets for oratory, podcast, and audiobook styles

### Added

- Cobra CLI framework with `elevenlabs` command and subcommands ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- `elevenlabs tts` command for generating speech from text files with SSML and emotion tag support ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- `elevenlabs ttsscript` command migrated from standalone `cmd/ttsscript` ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- YAML configuration file support (`--config`) for reusable TTS settings ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Built-in presets (`--preset`): `oratory`, `podcast`, `audiobook` with optimized voice settings ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Config save functionality (`--save-config`) to export current settings to YAML ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Voice settings CLI flags: `--stability`, `--similarity`, `--style`, `--speed` ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Automatic output format detection: `.mp3`, `.wav`, `.opus`, `.raw` based on format parameter ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- `ttsconfig` subpackage with `Config` and `VoiceSettings` types for programmatic TTS configuration ([`3137a14`](https://github.com/agentplexus/go-elevenlabs/commit/3137a14))
- `ttsconfig.Load()` and `ttsconfig.Save()` functions for YAML config file I/O ([`3137a14`](https://github.com/agentplexus/go-elevenlabs/commit/3137a14))
- `ttsconfig.Oratory()`, `ttsconfig.Podcast()`, `ttsconfig.Audiobook()` preset functions ([`3137a14`](https://github.com/agentplexus/go-elevenlabs/commit/3137a14))
- `ttsconfig.EstimateCredits()` and `ttsconfig.FormatDuration()` utility functions for credit calculation ([`3137a14`](https://github.com/agentplexus/go-elevenlabs/commit/3137a14))
- `ttsconfig.StripMarkup()` for removing SSML and emotion tags from text ([`3137a14`](https://github.com/agentplexus/go-elevenlabs/commit/3137a14))
- `--estimate` flag for credit estimation without calling API (shows words, duration, estimated credits) ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))

### Changed

- CLI uses `slog.Logger` for structured logging instead of `fmt.Printf` ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Default TTS model changed to `eleven_v3` for new CLI commands ([`6bb1bb9`](https://github.com/agentplexus/go-elevenlabs/commit/6bb1bb9))
- Migrated `cmd/ttsscript` standalone CLI to `elevenlabs ttsscript` subcommand ([`f5d5d6f`](https://github.com/agentplexus/go-elevenlabs/commit/f5d5d6f))

### Removed

- Standalone `cmd/ttsscript/main.go` (functionality moved to `elevenlabs ttsscript`) ([`f5d5d6f`](https://github.com/agentplexus/go-elevenlabs/commit/f5d5d6f))

### Dependencies

- Add `github.com/spf13/cobra` v1.10.2 for CLI framework ([`a345867`](https://github.com/agentplexus/go-elevenlabs/commit/a345867))
- Add `gopkg.in/yaml.v3` v3.0.1 for YAML config support ([`a345867`](https://github.com/agentplexus/go-elevenlabs/commit/a345867))

## [v0.7.0] - 2026-01-24

### Highlights

- WebSocket STT API migrated to scribe_v2_realtime with breaking changes to options and message formats
- WebSocket TTS stream completion detection via Done() channel for reliable end-of-stream handling

### Breaking

- `WebSocketSTTOptions` fields renamed: `SampleRate`/`Encoding` replaced with `AudioFormat`, `EnableWordTimestamps` replaced with `IncludeTimestamps`, `EnablePartials` removed (partials always enabled) ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTConnection.EndStream()` removed, use `Commit()` instead ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- Default model changed from `scribe_v1` to `scribe_v2_realtime` ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))

### Added

- `WebSocketSTTOptions.AudioFormat` field for specifying audio encoding (e.g., `pcm_16000`, `ulaw_8000`) ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTOptions.IncludeTimestamps` field for word-level timing ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTOptions.IncludeLanguageDetection` field for detected language in responses ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTOptions.CommitStrategy` field (`manual` or `vad`) for transcript commit control ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- VAD (Voice Activity Detection) settings: `VADSilenceThresholdSecs`, `VADThreshold`, `MinSpeechDurationMs`, `MinSilenceDurationMs` ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTConnection.Commit()` method for manual transcript commit ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTConnection.SendAudioWithCommit()` method for sending audio with optional commit ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketSTTConnection.SessionID()` method to retrieve server-assigned session ID ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- `WebSocketTTSConnection.Done()` channel for stream completion detection after `Flush()` ([`b4ac98d`](https://github.com/agentplexus/go-elevenlabs/commit/b4ac98d))
- OmniVoice TTS and STT provider conformance tests ([`707d6d2`](https://github.com/agentplexus/go-elevenlabs/commit/707d6d2))
- OmniVoice capabilities documentation with interface implementation matrix ([`e3d4815`](https://github.com/agentplexus/go-elevenlabs/commit/e3d4815))

### Changed

- WebSocket STT now uses query parameters for configuration instead of init message ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- Audio data sent as `input_audio_chunk` message type with `audio_base_64` field ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- Response message types: `partial_transcript`, `committed_transcript`, `committed_transcript_with_timestamps` ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- OmniVoice conversion utilities updated for new STT options structure ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))
- OmniVoice TTS provider uses `Done()` channel and inactivity timeout for reliable stream completion ([`b4ac98d`](https://github.com/agentplexus/go-elevenlabs/commit/b4ac98d))
- OmniVoice TTS `GetVoice()` returns `tts.ErrVoiceNotFound` for invalid voice IDs ([`b4ac98d`](https://github.com/agentplexus/go-elevenlabs/commit/b4ac98d))

### Dependencies

- Bump `github.com/agentplexus/omnivoice` from v0.2.0 to v0.3.0 ([`29ec159`](https://github.com/agentplexus/go-elevenlabs/commit/29ec159))

### Documentation

- `TRD_STT.md` technical requirements document for WebSocket STT migration ([`e3d4815`](https://github.com/agentplexus/go-elevenlabs/commit/e3d4815))
- WebSocket TTS stream completion behavior documentation with usage patterns ([`e3d4815`](https://github.com/agentplexus/go-elevenlabs/commit/e3d4815))
